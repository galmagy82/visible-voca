▸ API 키 발급 가이드

Gemini API 키 발급 방법

  1. Google AI Studio에 접속합니다.
  2. Google 계정으로 로그인합니다.
  3. "API 키 만들기" 버튼을 클릭합니다.
  4. 생성된 키를 복사하여 위에 붙여넣습니다.
Google AI Studio에서 결제 연동이 필요합니다.
검색 1회당 약 50원의 비용이 발생합니다.

OpenAI API 키 발급 방법

  1. OpenAI Platform에 접속합니다.
  2. OpenAI 계정으로 로그인합니다.
  3. "Create new secret key" 버튼을 클릭합니다.
  4. 생성된 키를 복사하여 위에 붙여넣습니다.
OpenAI에서 결제 연동이 필요합니다.
검색 1회당 약 50원의 비용이 발생합니다.
