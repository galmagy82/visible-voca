▸ API 키 발급 가이드

Gemini API 키 발급 방법

  1. Google AI Studio에 접속합니다.
  2. Google 계정으로 로그인합니다.
  3. "API 키 만들기" 버튼을 클릭합니다.
  4. 생성된 키를 복사하여 위에 붙여넣습니다.
Gemini API 키 결제 미설정 시
· 단어 검색: 하루 무료 제공량 안에서 사용 가능
· 이미지 생성: 사용 불가

Gemini API 키 결제 설정 시 (사용량에 따라 과금)
· 단어 검색: 제한 없음
· 이미지 생성: 사용 가능

OpenAI API 키 발급 방법

  1. OpenAI Platform에 접속합니다.
  2. OpenAI 계정으로 로그인합니다.
  3. "Create new secret key" 버튼을 클릭합니다.
  4. 생성된 키를 복사하여 위에 붙여넣습니다.
OpenAI에서 결제 연동이 필요합니다.
검색 1회당 약 50원의 비용이 발생합니다.
▸ API 키 안전 안내
입력하신 API 키는 이 기기의 브라우저(localStorage)에만 저장되며, 개발자의 서버로 전송되지 않습니다.
Google/OpenAI API 호출 시에만 해당 서비스로 직접 전송됩니다.
공용 PC에서는 사용 후 설정에서 키를 삭제해 주세요.
이전에 검색한 단어는 캐시에 저장되어 다시 검색해도 비용이 발생하지 않습니다.
캐시를 초기화하면 같은 단어를 다시 검색할 때 새로운 이미지가 생성되며 비용이 발생합니다.
